# 🦈 Shark Attack Exploratory Data Analysis (EDA)

## 📌 Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on global shark attack data.  
The goal is to clean the dataset, analyze patterns, and extract meaningful insights about shark attacks such as frequency, locations, activities, and outcomes.

---

## 🎯 Objectives
- Clean and preprocess raw shark attack data  
- Handle missing and inconsistent values  
- Perform exploratory data analysis  
- Identify patterns in shark attack incidents  
- Visualize trends using charts and graphs  

---

## 📁 Dataset
The dataset includes records of shark attack incidents with features such as:
- Date of incident  
- Location (country, region, etc.)  
- Activity during attack  
- Injury/fatality information  
- Shark species (when available)  

---

## 🛠️ Technologies Used
- Python 🐍  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## 📊 Key Analysis Performed
- Data cleaning and handling missing values  
- Distribution of shark attacks by country and year  
- Most dangerous activities  
- Fatal vs non-fatal attack comparison  
- Trend analysis over time
